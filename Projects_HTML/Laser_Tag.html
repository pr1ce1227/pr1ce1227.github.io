  • Custom Reciever Board
  • Custom Transmitter Board

Summary

This project is well known as the Junior Core Project for BYU Computer and Electrical Engineers. In this project we built a laser tag system using a custom Receiver and transmitter board. We then used the Zybo board to handle all the digital signal processing. Including an FIR and IIR filter to filter out the 10 different player signals. A player could select their frequency by flipping the switches on the zybo board. We then had to use our newly developed microcontroller skills to build the game and add audio to the different devices.

Overall this was a very challenging project that helped us practice all the skills we were currently learning in our classes. It's not very often you have something physical to show for your work in computer engineering bu this project was the acception. It was escpecially fun getting to play with a few round with everyone at the end of the year.
