Flappy Bird

Main Concepts

  • Advanced State Machines
  • Written in C only
  • Bare Metal Design
  • Control hardware through low level register access

Materials Used

  • Xilinx ZYBO Board

Summary

This was my first exposure to embedded systems and advanced state machines. This was a bare metal C project in which we had to control the hardware through low level register read and writes. The flappy bird game was my own design and completed as a free style project. In the game you can change the difficulty which is essentially just changing the clock driving the various state machines. In this game I had 2 main state machines one driving the realistic gravity affect on the ball the other driving the pipes along te screen. This was the first large scale project in C not in C++ and it gave me a new apreciation for the built in data structures that c++ offers.
